Community

We welcome you to participate in the Apache Grails® community. Before joining us, please review the Apache Community Code of Conduct.

Grails Mailing Lists

The Apache Grails community offers several mailing lists where you can connect with other users and contributors.

dev@grails.apache.org

For discussions about Grails framework development. Subscribe to this list if you contribute to Grails core or are interested in its evolution

To subscribe, send an email to
dev-subscribe@grails.apache.org.

Find the dev web archive here

users@grails.apache.org

For discussions related to using Grails in application development. Join this list if you are a Grails user looking for help, best practices, or community discussions.

To subscribe, send an email to
users-subscribe@grails.apache.org.

Find the user web archive here
