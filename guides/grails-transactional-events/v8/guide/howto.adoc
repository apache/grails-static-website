You can either type the code in this guide as you read or skip ahead and clone the finished sample:

[source,bash]
----
git clone -b grails8 https://github.com/grails-guides/grails-transactional-events.git
cd grails-transactional-events/complete
./gradlew integrationTest
----

`initial/` is the vanilla Grails 8 starter from link:https://start.grails.org[start.grails.org] with the default H2 in-memory database. `complete/` adds the `Customer`/`Order`/`AuditLog` domain model, the `OrderPlacedEvent` POGO, the `OrderService` publisher, the three listener beans, and the integration specs.
